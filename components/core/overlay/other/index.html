Other Overlay
Core

Used when you need to display any content over the main page.

Disable

You can disable styles and javascript using .overlay-disable.

SyntaxMixinVariantsDescription
Utility.overlay-disableoverlay-disable !importantresponsiveReset overlay styles and javascript
#demo--overlay-nested-disable-toggle { /*@apply md:hidden !important;*/ @apply hidden !important; } #demo--overlay-nested-disable { /*@apply md:overlay-disable !important;*/ @apply overlay-disable !important; }
import { Xt } from 'xtendui' Xt.mount.push({ matches: '#demo--overlay-nested-disable-toggle', mount: object => { // event const click = () => { document.querySelector('#demo--overlay-nested-disable').dispatchEvent(new CustomEvent('on.trigger.xt')) } object.addEventListener('click', click) }, })

Preset

You can overlay that covers the screen with .overlay-screen, remember to use tailwind classes (e.g. align-items, justify-content) on .card.

SyntaxMixinVariantsDescription
Utility.overlay-screenoverlay-screenresponsiveOverlay that covers the screen

To create an overlay position with .overlay-screen, you can also use tailwind classes (e.g. max-width, margin) on .overlay-container.

Close fixed

You can have the .btn-close with fixed z-top to have it always visible also when scrolling.
