Block

Sequential .card-block stack vertically. Use tailwind classes to assign styles (e.g.: border-radius, background, spacing)

SyntaxMixinDescription
Component.card-blockcard-blockCard stack vertical

Group

To stack .card-block horizontally wrap them inside .card-group.

SyntaxMixinVariantsDescription
Util.card-groupcard-groupresponsiveCard stack horizontal wrapper

The horizontal size of .card-block is automatic, but you can specify it with tailwind width.
