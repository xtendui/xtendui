Customization

You can customize this component inside tailwind.config.js, check xtendui/src/addons/hero.css.js for default values.

module.exports = {
  theme:{
    xtendui: {
      hero: {
        utility: theme => ({
          // modify utility
        }),
      },
    },
  },
}

Usage

Use component classes and tailwind classes to create a hero.

Content

Use tailwind classes to customize content.

Use component classes to add hero shadow.
