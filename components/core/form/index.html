Form
Core

Form styles with multiple layouts, custom checkbox/radio, javascript validation, and more.

Installation

Import the component javascript file.

import 'xtendui/src/core/form'

Customization

You can customize this component inside tailwind.config.js, check xtendui/src/core/form.css.js for default values.

module.exports = {
  theme:{
    xtendui: {
      form: {
        component: theme => ({
          // modify components
        }),
      },
    },
  },
}

Usage

Use this code to create a form items.

For aria purpose label form controls on input, select, textarea, we use aria-label="Label".

Check subpage to customize content and customize layout.
