Validation

Html validation and Constraint Validation API.

It automatically runs on form submit and automatically scrolls on the first invalid form item.

Use component classes to style validation.

SyntaxMixinDescription
Componentform[novalidate]NOT POSSIBLEDo not validate form
Component.form-validform-validValidation valid class
Component.form-invalidform-invalidValidation invalid class
Component.form-label-requiredform-label-requiredLabel required with asterisk
