Badge
Core

Component for showing short and relevant tags.

Customization

You can customize this component inside tailwind.config.js, check xtendui/src/core/badge.css.js for default values.

module.exports = {
  theme:{
    xtendui: {
      badge: {
        component: theme => ({
          // modify components
        }),
      },
    },
  },
}
