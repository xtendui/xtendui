Api Toggle
Core

Toggle is the main js component that toggle classes, it's inherited by many other js components.

Initialization

Initialize automatically within markup with [data-xt-toggle="{ <options> }"] on the object (the DOM element you assigned the component).

Or initialize with javascript.

let self = new Xt.Toggle(document.querySelector('#my-object'), {
  // options
})

Or inizialize with mutation observer.

Xt.mount.push({
  matches: '#my-object',
  mount: object => {
    // init

    let self = new Xt.Toggle(object, {
      // options
    })

    // unmount

    return () => {
      self.destroy()
      self = null
    }
  }
})

Listen

Listen to events on DOM elements.

const el = document.querySelector('#my-element-or-target')

const eventOn = e => {
  if (e.target === el) {
  // logic
  }
}

el.addEventListener('on.xt', eventOn)

Listen to events delegation with useCapture this way:

let object = document.querySelector('#my-object')
let self = Xt.get('xt-toggle', object)

const eventOn = e => {
  const element = e.target
  // useCapture delegation
  if (self.elements.includes(element)) {
    // logic
  }
  if (self.targets.includes(element)) {
    // logic
  }
}

object.addEventListener('on.xt', eventOn, true)

You can get activated elements or targets with hasCurrent.

self.elements.filter(x => self.hasCurrent(x))
self.targets.filter(x => self.hasCurrent(x))
