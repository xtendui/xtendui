Content
Media
Design

Wrapper to have media with aspect ratio, also on page load, useful also for advanced animations.

Variant

When images are loading the .xt-media-container background is shown.

Use tailwind classes to assign variant (e.g. background-color).

Responsive

Use tailwind classes .relative on .xt-media to assign relative position. The result is a responsive media.

SyntaxMixinDescription
Tailwind.relativerelativeRelative size

Aspect ratio

Assing padding-bottom on .xt-media-container to the aspect ratio you desire.

Or use component utility .xt-ratio-<value> on .xt-media-container to make aspect ratio images.

SyntaxMixinVariantsDescription
Utility.xt-ratio-<value>ratio-<value>responsiveApply aspect ratio, can be 15, 37.5, 50, 56.2, 75, 100, 125, 150

Object fit

Use tailwind classes to assign object fit. (e.g. object-position).

You can also use video or iframe for media.

Height

Use tailwind classes to assign height (e.g. height).

You can also use min-height: <Value> to have a min height (use narrow screen to see it in action).
