Ripple Animation
Addons

Material ripple effect on user interaction.

Installation

Import the component javascript file.

import 'xtendui/src/addons/ripple'

Customization

You can customize this component inside tailwind.config.js, check xtendui/src/addons/ripple.css.js for default values.

module.exports = {
  theme:{
    xtendui: {
      ripple: {
        component: theme => ({
          // modify components
        }),
      },
    },
  },
}

Usage

Use this code to create a ripple.

Initialization

Initialize automatically within markup with [data-xt-ripple="{ <options> }"] on the object (the DOM element you assigned the component).

Or initialize with javascript.

let self = new Xt.Ripple(document.querySelector('#my-object'), {
  // options
})

Or inizialize with mutation observer.

Xt.mount.push({
  matches: '#my-object',
  mount: object => {
    // init

    let self = new Xt.Ripple(object, {
      // options
    })

    // unmount

    return () => {
      self.destroy()
      self = null
    }
  }
})
