Use overlay when you need to display any content over the main page, comes in different options and factors.

Disabled

If you want to disable the component use the options.disabled.

SyntaxDefault / ArgumentsDescription
Optiondisabled:BooleanfalseDisable the component

Match Media

You can add additional options that gets added on match media query. You can use different queries with different and nested options.

SyntaxDefault / ArgumentsDescription
Optionmatches:ObjectfalseAdd additional options on match media query

Unclosable

You can have an unclosable overlay with closeDeep: false and closeInside: false.
