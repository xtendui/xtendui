Sticky elements with complex interactions.

Installation

Import the component javascript file.

import 'xtendui/src/core/sticky'

Customization

You can customize this component inside tailwind.config.js, check xtendui/src/core/sticky.css.js for default values.

module.exports = {
  theme:{
    xtendui: {
      sticky: {
        component: theme => ({
          // modify components
        }),
      },
    },
  },
}
