Usage

Use this markup to create a fade.

You can initialize scroll by javascript omitting [data-xt-scroll].

  
    let self = Xt.init('xt-scroll', document.documentElement, {
      // option
    });
    self.unmount = function() {
      self = null;
    };
It's recomended to animate the scroll elements with animation instead of transition because of this bug.

Options

Mode

You can set scroll mode with less mixins (@TODO link less mixins).

Default mode infinite
&:not(.scroll--once)
&:not(.scroll--visible)
It's possible to specify different modes with media query mixins (@TODO link media queries mixins) for responsive
.scroll--block

Direction and Distance

Animate with inverse .inverse or self.detail.inverse, use "distance": Number activation and deactivation distance. For percent use "distance": "Number%"

Timing

You can use a function for "delayOn" and "delayOff" like this: function(current, total) {return current * 75}. You can also use a string "return current * 75".

Parallax

Use events 'change.xt.scroll' inside you can use the values of self.detail.ratio self.detail.ratioInverse self.detail.ratioDouble
