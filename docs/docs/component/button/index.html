Usage

Use .btn to create a button.

Variables

Read the documentation of component's variables.

Options

Variants

You need to specify a variant on each .btn.

.btn-primary .btn-primary-empty .btn-secondary .btn-secondary-empty .btn-link

Sizes

Optional classes for assigning size. Sizes uses em padding that depends on font size.

.btn-tiny .btn-small .btn-medium .btn-big .btn-giant

Special

Special optional modifiers classes.

.btn-icon has special padding for icons.

.btn-nodesign .btn-squared .btn-tall .btn-short .btn-wide .btn-narrow .btn-icon .btn-none .btn-none-x .btn-none-y .btn-transform

Custom

.btn-caret .btn-close

Links

On containers use .inverse to have white links.

Use the mixins .link-underline() or .link-none() to add or remove text-decoration: underline; on links and buttons.

a .alink .inverse
